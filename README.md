⚡ 闪传 (Flashdrop)

简单、快速的文件传输。

无需登录，即用即走。

闪传 (Flashdrop) 是一款基于 Node.js 和 React 构建的极简私有云文件中转站。

它拥有现代化的毛玻璃拟态 UI (Glassmorphism)、智能的存储空间调度以及金融级的“阅后即焚”物理销毁机制，专为追求高隐私、高效率、免打扰的临时跨设备文件/文本传输而设计。

✨ 核心特性

🔐 极致隐私保障多重即焚机制：生命周期超时物理粉碎；多独立取件码耗尽粉碎；防偷窥退出即焚（用户关闭或刷新下载页即发送底层销毁信标）。

灵活有效期：支持自定义 1小时、10小时、1天、3天、7天 文件留存时间。

裂变取件码：一次上传可生成 1~5 个独立的两位置取件码（00~99），共享同一份物理文件，互不干扰。

🚀 强悍的传输与调度防爆拦截：单文件硬性限制最高 5GB。

流式下载：采用原生流媒体传输，百兆/GB级大文件下载满速且不吃服务器内存。

智能容量守护：后台自动执行 20GB 总空间容量巡检，一旦超额自动释放最旧文件，永远告别硬盘撑爆。

🎨 前卫的美学与交互高级晶态 

UI：动态骨架屏防白屏秒开，取件码金字塔智能排版，原生动画过渡。

全键盘支持：提取页面全局监听物理键盘输入（支持 0~9 键入、Backspace 删除、Esc 清空）。

纯文本分享：不仅支持文件互传，更支持大段文本数据的跨端复制。

🛠️ 技术栈前端：React 18 (CDN直出) + TailwindCSS + FontAwesome + HTML5后端：Node.js + Express + Multer (流上传)

存储：基于本地文件系统的零依赖轻量化内存引流映射（无需额外安装 MySQL 或 Redis）

🚀 部署指南

1. 基础安装确保您的服务器已安装 Node.js (建议 v16 及以上版本)。克隆或下载本项目到服务器后，进入项目根目录：# 安装依赖
npm install

2. 启动服务 (默认运行在 3000 端口)
npm start

如果您希望服务在后台持久运行，推荐使用 pm2：npm install pm2 -g
pm2 start server.js --name "flashdrop"

3. ⚠️ 极其关键的 Nginx 配置 (生产环境必看)如果您使用 Nginx 反向代理该服务（绑定域名），Nginx 默认会拦截所有大于 1MB 的文件，导致大文件上传瞬间报错。

您必须在站点的 Nginx 配置文件（nginx.conf 的 server 块内）加入以下配置，以放行最高 2GB 的文件：server {
    listen 80;
    server_name yourdomain.com;

    # 【重要】允许最高 5GB 的上传限制
    client_max_body_size 5120M;

    location / {
        proxy_pass [http://127.0.0.1:3000](http://127.0.0.1:3000);
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    }

📖 使用说明发送端：访问首页，点击“发送”。

拖拽文件或输入想分享的文本。

设置“保存时间”和“生成数量”，点击获取。

将生成的 2 位数专属取件码（如 59）告知接收方。

接收端：访问首页，点击“接收”。

在键盘直接敲击或点击屏幕输入 2 位提取码。

若只有单一文件且无留言，系统将静默自动下载；若包含多文件/留言，系统将展示详细的下载确认页。

销毁：接收方提取完成并离开页面后，文件将从服务器的硬盘中永久物理删除，无法恢复。
